# CodeMentor 🚀

CodeMentor is an AI-assisted developer support platform designed to guide programmers through their coding workflow.  
It acts like a digital mentor that helps users **plan, analyze, and manage their coding tasks effectively**.

This project was developed as part of a **GenAI Hackathon**, focusing on improving developer productivity and learning.

---

## 🌟 Project Overview

CodeMentor provides an interactive web interface where users can:

- Login securely
- Access a dashboard for navigation
- Plan coding tasks and project workflows
- Analyze code or development approaches
- Manage their development process efficiently

The platform simulates the role of a **mentor for developers**, guiding them through structured planning and analysis.

---

## 🎯 Problem Statement

Many beginner developers struggle with:

- Planning coding tasks
- Structuring projects
- Understanding how to approach problem solving
- Organizing development workflow

CodeMentor aims to solve this by providing a **guided environment that supports planning and analysis**, similar to how a mentor assists a student.

---

## ⚙️ Features

### 🔐 Authentication System
- User login interface
- Authentication handling
- Secure access to dashboard

### 📊 Dashboard
- Central hub of the application
- Provides navigation to all features
- Displays user interaction flow

### 📝 Planner Module
- Allows users to plan coding tasks
- Helps structure development workflow
- Supports step-by-step problem solving

### 🔍 Analyzer Module
- Analyzes inputs related to coding tasks
- Helps users evaluate their development approach
- Provides insights for improvement

## 🔄 Workflow

1. User opens the application
2. User logs in through the authentication system
3. User enters the dashboard
4. User can access:
   - Planner module to organize coding tasks
   - Analyzer module to analyze problem-solving approaches
5. The system provides structured guidance similar to a coding mentor
